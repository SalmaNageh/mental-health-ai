 MindCare AI
MindCare AI is an AI-powered mental health analysis platform built using Machine Learning, NLP, Streamlit, and SQLite.
The system analyzes user text and predicts emotional or mental health states such as:
- Anxiety
- Depression
- Stress
- Suicidal
- Normal
The project includes authentication, analytics dashboards, prediction history, admin controls, and interactive visualizations.

 Features
  Login & Signup System
  Admin Dashboard
  NLP + Machine Learning Prediction
 Analytics Dashboard
Confusion Matrix & Model Performance
SQLite Database Integration
User Prediction History
 Modern UI (Glassmorphism + Animations)
Dark / Light Mode
Streamlit Cloud Deployment

🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- NLP (TF-IDF + LinearSVC)
- SQLite
- Pandas
- Matplotlib
- Seaborn
- WordCloud
